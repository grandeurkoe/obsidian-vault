# Arrays or Lists

## Time Complexity

1. LOOKUP - O(1)
2. PUSH - O(1)
3. INSERT - O(n)
4. DELETE - O(n)

## What are Array good for?

1. Fast lookup.
2. Fast push/pop (last item in a list or array).
3. Ordered.

## What are Array bad for?

1. Slow inserts.
2. Slow deletes.
