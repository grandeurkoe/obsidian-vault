# Dictionaries

Create a dictionary -
```python
{Key1 : Value1 , Key2: Value2}
```

Get the value given a key -
```python
dictionary_name[key1]
```

Adding a new key-value pair -
```python
dictionary_name[key1] = value1
```

Create an empty dictionary -
```python
# This can also be used to wipe a dictionary.
dictionary_name = {}
```
