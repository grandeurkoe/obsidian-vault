# Nested List

Nested List is a List within a List.

Nested list would look something like this :
```python
my_list = [[1, 2, 3], [4, 5, 6]]
```

Another way to implement nested loop would be :
```python
list1 = ["item1", "item2", "item3"]
list2 = ["item1", "item2", "item3"]
list3 = [list1, list2]
```
