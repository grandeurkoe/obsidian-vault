HTML Heading Element looks like this:

```html
	<h1>Hello World!</h1>
```

Here,
- `<h1>` is the opening tag.
- `</h1>` is the closing tag.
- `Hello World!`is the content of the `h1` element.

## `<Tag>` vs. Element

`<h1>` and `</h1>` are tags i.e., anything inside the angle brackets.
