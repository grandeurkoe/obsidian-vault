# Mathematical Operations

Python supports the following Mathematical operation -

1. n1 + n2 : Add n1 and n2.
2. n1 - n2 : Subtract n2 from n1.
3. n1 * n2 : Multiply n1 and n2.
4. n1 / n2 : Divide n1 by n2.
5. n1 % n2 : Returns the remainder when n1 is divided by n2.
6. n1 ** n2 : Raise n1 to an exponent of n2.
